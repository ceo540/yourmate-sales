YOURMATE INTERNAL SYSTEM

유어메이트 운영 시스템

리드부터 계약·실행·정산까지, 사무실에서 일어나는 모든 일을 AI(빵빵이)와 함께 정리·실행하는 자체 시스템.

사용자 유어메이트 임직원 접속 yourmate-system.vercel.app 구성 Next.js + Supabase + Dropbox + Anthropic Claude

⭐ 우리가 만드는 것

"AI가 사무업무를 직접 실행하고, 사람은 매니징과 현장에 집중한다." 새 기능 검토 시 항상 묻는 질문 — "이거, AI가 직접 할 수 있나?"

왜 이 시스템을 만드나

중소기업이 흔히 겪는 "여러 도구(노션·엑셀·카톡·이메일·드롭박스) 사이를 사람이 손으로 옮기며 일하는" 비효율을 없애는 것. 한 곳에서 다 보고, AI가 정리·생성·전달까지 해준다.

📋 메뉴 카탈로그

사이드바에 보이는 11개 메뉴. 각자 어떤 일을 하는 화면인지.

🏠

홈 (대시보드)

로그인 직후 첫 화면. 오늘 할 일·임박 리드·진행 중 프로젝트 한눈에.

📥

리드

문의가 들어와서 아직 계약은 안 된 상태. 채널톡·전화·이메일 들어온 거 다 모임. 빵빵이가 견적 초안 자동 생성.

유입→회신대기→견적발송→조율중→진행중→완료
◈

프로젝트

실제 진행 중인 일들. 한눈에·자세한 개요, 협의해야 할 내용(클라이언트/내부/외주사 3분류), 할 일, 캘린더 일정, 메모, 빵빵이 협업.

63건 진행 중
📜

계약 (sales)

매출 건. 계약→착수→선금→중도금→완수→계산서발행→잔금 7단계. 사업부별 통계, 한도 현황, 정렬·필터.

⬡

서비스

교구대여(rentals), SOS, 사업부별 라인업. 캘린더 멀티라인 표시, 신규 등록 picker.

🗂️

고객

기관(customers) + 담당자(persons) 분리 관리. 같은 기관 다른 부서 산하 매핑(예: 미래재단 산하 청소년수련관) 가능.

📅

캘린더

모든 프로젝트 일정 한 화면. 멀티라인 셀(건명/고객/날짜). Google Calendar 연동(WIP).

📊

재무

관리자 전용. 월별 매출·미수금·외주비·마진. 사업부별 분리.

관리자 전용
👥

팀 / HR / 출근

직원 명단·역할·연락처·소속 부서. 출근 체크는 별도 시스템(yourmate-attendance) 연동.

📢

공지

회사 공지사항. 모든 직원 즉시 확인.

✅

업무 (tasks)

모든 프로젝트의 할 일을 사람별로 모아 보기. 데드라인·우선순위·상태·생성순 정렬.

⚙️

관리

관리자 영역. 팀 관리, 직원, 사업자, 권한, 알림 설정, API 사용료(빵빵이 비용 추적).

관리자 전용

🤖 빵빵이 — AI 도우미

사이드바 우측에 항상 떠있는 AI 도우미. 자연어로 시키면 시스템 곳곳을 직접 조작·정리·생성한다. 모델: Claude Sonnet 4.6.

📝 새 계약 / 새 리드 등록

"OO학교 행사운영 2천만원 계약 체결했어" → DB에 자동 등록 + Dropbox 폴더 생성 + 프로젝트 자동 생성.

create_sale create_lead

🔄 기존 건 업데이트

"포천시청소년재단 견적 보냈어" → 상태·내용 자동 변경 + 소통 로그 자동 추가.

update_sale_status update_lead add_communication_log

💬 자연어 질문

"이번달 매출 얼마야?" "리마인드 임박한 리드 있어?" "미수금 현황 알려줘" 등 자연어로 즉시 답변.

🧠 프로젝트 분석

프로젝트 페이지에서 "한눈에 자동 생성", "협의 일괄 분석", "자세한 개요 다시 작성" 등.

regenerate_overview regenerate_pending_discussion regenerate_short_summary

📋 견적 자동 생성

"이 리드로 견적 뽑아줘. 항목은 사회자 30만원, 음향 200만원, 부가세 포함" → HTML 견적서 + Dropbox 자동 저장.

create_quote update_quote list_quotes

📋 할 일 관리

"내일까지 견적서 보내라고 메모 남겨줘" → 할 일 자동 추가. 완료/삭제도 자연어로.

create_task complete_task update_task

📁 Dropbox 직접 조회

"이 프로젝트 견적 PDF 뭐 있는지 확인해" → 폴더 목록 + 파일 직접 읽기.

list_project_files read_project_file

👤 고객 검색·생성

"지난번 진행한 OO학교 정보 찾아줘" → DB 검색 + 즉시 결과. 새 기관도 빵빵이가 직접 등록.

search_customers quick_create_customer

⚡ 자동화 — 사람 손 안 가는 일들

사람이 매번 챙기지 않아도 시스템이 알아서 도는 일.

매일 아침 09시 → 결제일 임박/지연 프로젝트 자동 검색 → 담당자에게 SMS·이메일 자동 발송 → 미수금 손실 0
채널톡으로 문의 도착 → 채널톡 빵빵이가 1차 응대 → 의도 파악 후 리드 자동 생성 → 담당자 알림 + 후속 사람 응대
계약 → 진행중 변경 → Dropbox에 프로젝트 폴더 자동 생성 → 템플릿 폴더 구조 복사 → 바로 협업 가능
